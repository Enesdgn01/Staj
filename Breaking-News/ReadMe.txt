Animasyon Adı: @keyframes ile tanımlanan animasyonun adı.
Süre: Animasyonun ne kadar süreceği. Genellikle saniye (s) veya milisaniye (ms) cinsinden ifade edilir.
Zamanlama Fonksiyonu: Animasyonun hızını ve zamanlamasını kontrol eder (örneğin, linear, ease-in, ease-out).
Döngü Sayısı: Animasyonun kaç kez tekrar edeceğini belirler (infinite sürekli tekrar eder).
@keyframes animation-name {
  0% {
    /* Başlangıç durumunu tanımlar */
  }
  50% {
    /* Orta durumunu tanımlar (isteğe bağlı) */
  }
  100% {
    /* Bitiş durumunu tanımlar */
  }
}
